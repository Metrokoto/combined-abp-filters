# Combined ABP Filters
This repository is for my personal ABP blocklist used across all of my deployments of it. 
## But why combine them...?
There are many reasons why I choose to do this, however, this repository came about after I discovered having too many filters added to Cromite's fork of it caused some rules to simply not apply.

As a result, I set out to find a solution, and I found that combining them into one filter subscription fixes the issue.
